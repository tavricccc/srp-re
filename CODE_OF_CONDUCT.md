# Code of Conduct / 社群行為準則

## 繁體中文

我們致力於提供友善、安全且包容的協作環境。參與者應尊重不同經驗與觀點、提供具體且建設性的回饋、聚焦議題本身，並保護他人的個資與安全。

不可接受的行為包括騷擾、歧視、威脅、性化言語或圖像、人身攻擊、惡意公開個資、持續打擾，以及鼓勵他人從事上述行為。本準則適用於 issue、pull request、review、discussion 及代表專案參與的其他公開或私人空間。

維護者可編輯或移除不當內容、警告參與者、暫時或永久限制互動。若需回報，請透過 repository 擁有者 GitHub 個人檔案所列的私人聯絡方式，附上連結、時間與必要證據；維護者會盡可能保護回報者隱私。

## English

We are committed to a welcoming, safe, and inclusive collaboration environment. Participants should respect different experiences and viewpoints, offer specific constructive feedback, focus on the issue, and protect others' privacy and safety.

Unacceptable behavior includes harassment, discrimination, threats, sexualized language or imagery, personal attacks, malicious disclosure of private information, sustained disruption, or encouraging any of these actions. This policy applies to issues, pull requests, reviews, discussions, and other public or private spaces where someone represents the project.

Maintainers may edit or remove inappropriate content, warn participants, or temporarily or permanently restrict interaction. Report incidents privately using a method listed on the repository owner's GitHub profile, including links, time, and necessary evidence. Maintainers will make a reasonable effort to protect reporter privacy.
