# Security policy / 安全政策

## 繁體中文

請勿公開回報漏洞。請依[安全與隱私模型](docs/security.md#回報安全問題)的私下流程聯絡 repository 擁有者，並避免存取非你擁有的資料或中斷正式服務。

本專案目前僅維護預設分支的最新版本，沒有承諾回覆時限或漏洞獎勵計畫。

## English

Do not disclose vulnerabilities publicly. Follow the private process in the [security and privacy model](docs/en/security.md#reporting-a-vulnerability), and do not access data you do not own or disrupt production.

The project currently supports only the latest version on the default branch and promises neither a response SLA nor a bounty.
